Page Repair options

The extension fixes heading structure and landmarks entirely on your device. Labeling unnamed buttons uses the Anthropic API, which requires an API key. Only the HTML snippets of unnamed controls are sent — never the full page.

Option A: your own API key

Calls go directly from your browser to Anthropic. Nothing passes through our servers — you can audit exactly what is sent. If both options are filled in, your own key is used.

Option B: prepaid credits

No API account needed — labeling runs through the Page Repair service. One credit repairs one page; top up whenever you run out. No subscription, nothing renews on its own.
