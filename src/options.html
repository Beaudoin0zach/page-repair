Page Repair options

Heading and landmark repair runs entirely on your device, free, with no setup. Labeling unnamed buttons uses the Anthropic API and needs one of the two options below. Only the HTML snippets of unnamed controls (plus the page title and address, without query strings) are sent — never the full page.

Option A: prepaid credits

No API account needed — labeling runs through the Page Repair service. One credit repairs one page; top up whenever you run out. Prepaid only: nothing renews on its own.

Option B: your own Anthropic API key

Calls go directly from your browser to Anthropic. Nothing passes through our servers — you can audit exactly what is sent. If both options are filled in, your own key is used.
